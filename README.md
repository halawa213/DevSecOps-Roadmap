## Roadmap Overview

This roadmap is split into **9 phases over 48 months**, targeting a CS student pace of 10–15 hours per week.

| # | Phase | Duration | Key Focus |
|---|-------|----------|-----------|
| `00` | **Security Foundations** | M 1–4 | Security mindset, CIA triad, STRIDE, Linux basics, Python scripting |
| `01` | **Linux & OS Mastery** | M 5–8 | System hardening, kernel security, eBPF, shell scripting, auditd |
| `02` | **Network Security** | M 9–12 | TCP/IP internals, Wireshark, Nmap, WireGuard, firewall engineering |
| `03` | **Cloud Core & SecOps** | M 13–18 | AWS fundamentals, IAM, CSPM, Prowler, GuardDuty — **AWS SAA** cert |
| `04` | **Container & K8s Security** | M 19–24 | Docker hardening, Kubernetes RBAC, Falco, OPA/Gatekeeper — **CKA** cert |
| `05` | **DevSecOps Pipelines** | M 25–30 | Shift-left security, SAST/DAST/SCA, GitHub Actions, secrets management |
| `06` | **Threat Detection & SIEM** | M 31–36 | Sigma rules, ELK stack, incident response, threat hunting — **AWS Security Specialty** |
| `07` | **Zero Trust Platform** | M 37–42 | BeyondCorp, SPIFFE/SPIRE, mTLS, policy-as-code — **CKS** cert |
| `08` | **Red Team & Compliance** | M 43–48 | PACU, pen testing, SOC 2 / ISO 27001, OWASP LLM Top 10 — **Security+** |

## Features

- **Interactive Dashboard** — live progress tracker, current phase, ETA, and next task at a glance
- **Checkboxes with persistence** — check off tasks and your progress is saved locally
- **Search & Filter** — instantly find any skill, tool, or project across all phases
- **Dark / Light mode** toggle
- **Collapsible phases** — focus on what's relevant, collapse the rest
- **Per-phase progress bars** — see exactly how far you are in each phase
- **Mark phase complete** — one-click phase completion with visual feedback

## Who Is This For?

This roadmap is built for:

- CS students (Year 1–2) who want to enter the security field with a clear, structured plan
- Junior developers looking to pivot into DevSecOps or Cloud Security
- DevOps engineers wanting to add a security specialization
- Anyone targeting EU/DACH security roles — NIS2 compliance demand is driving significant hiring
- Egyptian engineers targeting remote EU/US roles or local banking & fintech security positions

This roadmap is **not** a generic "learn hacking" guide. It's an engineering career path focused on building, securing, and defending production systems.

## Why DevSecOps & Cloud Security?

| Factor | Why It Matters |
|--------|----------------|
| **AI Resistance** | Threat modeling, pen testing, compliance judgment, and security architecture cannot be automated |
| **Salary Premium** | Security engineers command a 20–30% premium over general DevOps roles |
| **NIS2 / DACH Demand** | EU's NIS2 directive is creating a wave of mandatory Cloud Security hiring across Germany, Austria, and Switzerland |
| **Egypt Market** | CIB, Vodafone VOIS, EG-CERT, and major Egyptian banks are all scaling their security engineering teams |
| **Global Mobility** | This cert stack (SAA + CKA + AWS Security Specialty + CKS + Security+) opens doors to Germany Blue Card, UAE, Netherlands, and Canada |

- **Security Consultant / Founder** — serve Egyptian SMEs, fintech, and financial institutions
- **Relocation** — DACH, UAE, Netherlands, or Canada with full cert stack + experience
